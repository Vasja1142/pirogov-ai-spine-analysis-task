# Тестовое задание Pirogov.AI

**Позиция:** Стажёр-инженер по машинному обучению.

## Цель задания

Целью данного задания является проверка ваших практических навыков в области компьютерного зрения, работы с медицинскими данными и способности к самостоятельному решению комплексных задач. Вы продемонстрируете умение находить и подготавливать данные, обучать современные нейросетевые модели, а также проводить базовый медицинский анализ.

## Описание задачи

Вам необходимо разработать прототип системы для автоматического анализа рентгеновских снимков позвоночника. Задача системы — **инстанс-сегментация, либо детекция позвонков**: нахождение на изображении каждого позвонка и определение его порядкового номера (например, C3, T5, L4) для шейного, грудного и поясничного отделов. Помимо выполнения основной задачи также требуется теоретическое изучение и формирование насмотренности по патологиям позвоночника на примере остеохондроза.

## Этапы выполнения

### 1. Поиск и подготовка данных
*   Самостоятельно найдите открытые наборы данных для рентгенографии позвоночника, которые содержат разметку для детекции или сегментации позвонков. В качестве источника данных используйте площадки для работы с данными, такие как Roboflow, Kaggle и другие.
*   Изучите структуру данных, форматы разметки (JSON, XML, маски) и приведите их к единому формату, пригодному для обучения.
*   Напишите скрипты для загрузки и предобработки изображений (нормализация, аугментация данных и т.д.).

### 2. Обучение модели инстанс-сегментации, либо детекции
*   Выберите и реализуйте современную архитектуру нейронной сети для задачи инстанс-сегментации, либо детекции.
*   Обучите модель на найденном наборе данных.
*   Оцените качество модели на тестовой выборке с помощью стандартных метрик: mAP, mAR для детекции и mIoU или Dice score для сегментации.

### 3. Изучение медицинской составляющей и классификация остеохондроза
*   **Самостоятельное обучение:** Вам необходимо изучить рентгенологические признаки остеохондроза позвоночника. Рекомендуется использовать медицинские учебные пособия, научные статьи и ресурсы для врачей-рентгенологов.
*   **Ключевые признаки для поиска:**
    *   Снижение высоты межпозвоночного диска
    *   Субхондральный склероз
    *   Остеофиты
    *   Спондилолистез

## Требования к результату

Вам необходимо предоставить:

**1. Код:**
*   Jupyter Notebook или Python скрипты, содержащие весь ваш пайплайн: от загрузки данных до инференса.
*   Код должен быть читаемым, хорошо прокомментированным и размещен на GitHub.

**2. Отчет (в виде `README.md` на GitHub или отдельного документа):**
*   Описание найденных датасетов и ссылки на них.
*   Обоснование выбора архитектуры модели.
*   Примеры результатов работы модели (визуализация с наложением масок/контуров и номерами позвонков).
*   Анализ ошибок модели: на каких позвонках или случаях она ошибается чаще всего и почему (с вашей гипотезой).
*   План по развитию проекта для определения патологии - остеохондроз.
*   Выводы по проекту.
